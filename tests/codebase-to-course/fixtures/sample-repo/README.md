# Sample Counter App

This tiny repo exists only to drive the `codebase-to-course` test contract.

- `src/index.js` wires a click handler to a button.
- `src/router.js` decides which handler runs for an action.
- `src/store.js` persists a click count in memory.

The generated course does not need to explain this repo deeply. The fixture only
needs to be realistic enough for `init-course.sh` to accept a source project and
create a course skeleton.
